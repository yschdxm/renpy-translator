# 🎮 Ren'Py 游戏翻译工具

一个基于 NiceGUI 的 Ren'Py 游戏汉化工具，支持 AI 翻译和手动翻译，让游戏汉化更简单。

## ✨ 功能特点

### 📁 项目管理
- 创建翻译项目，绑定游戏目录
- 自动解包 `.rpa` 文件
- 自动反编译 `.rpyc` 文件
- 使用 Ren'Py SDK 生成标准翻译文件

### 👤 人名翻译
- 自动提取游戏角色名
- 支持 AI 批量翻译人名
- 支持手动编辑人名翻译
- 人名词典自动保存

### 📊 人物分析
- 分析角色台词，生成人物特征
- 包含性格、说话风格、翻译建议等
- 翻译时自动注入角色特征

### 📝 字符串翻译
- 翻译菜单选项、按钮等 UI 文字
- 支持搜索和筛选
- 支持批量翻译

### 💬 对话翻译
- 翻译游戏对话和旁白
- 支持搜索、筛选、按角色过滤
- 支持上下文显示
- AI 翻译时自动注入角色特征

### 📦 游戏导出
- 一键导出翻译后的游戏
- 自动配置中文字体
- 自动添加语言选择界面
- 导出目录：`projects/项目名/output/`

## 🚀 快速开始

### 安装

```bash
# 克隆项目
git clone https://github.com/yschdxm/renpy-translator.git
cd renpy-translator

# 安装依赖
uv sync
```

### 准备字体

将一款开源中文字体放入 `fonts/` 目录：

```bash
# 示例：下载 Noto Sans SC 字体
mkdir fonts
cd fonts
# 从 https://fonts.google.com/noto/specimen/Noto+Sans+SC 下载
# 将 .otf 或 .ttf 文件放入此目录
```

```
renpy-translator/
├── fonts/
│   └── NotoSansSC-Regular.otf
```

### 配置 Ren'Py SDK

1. 下载对应平台的 [Ren'Py SDK](https://www.renpy.org/latest.html)
2. 解压到项目目录（如 `renpy-8.5.3-sdk`）
3. 在工具的"模型配置"页面设置 SDK 路径

**跨平台兼容**：SDK 在不同平台完全兼容：
- Windows: 自动使用 `renpy.exe`
- macOS/Linux: 自动使用 `renpy.sh`

### 配置 AI 模型

1. 在"模型配置"页面添加 AI 模型
2. 支持 OpenAI 兼容接口（如 DeepSeek、Claude 等）
3. 设置 API 地址、Key、模型名称

### 启动

```bash
uv run python run.py
```

访问 http://localhost:7860

## 📖 使用流程

### 1. 创建项目
- 输入项目名称
- 选择游戏目录
- 选择 AI 模型
- 点击"创建项目"

### 2. 人名翻译
- 切换到"人名翻译"标签页
- 点击"翻译全部人名"或逐个翻译
- 检查并修正翻译结果

### 3. 人物分析
- 切换到"人物分析"标签页
- 点击"分析所有角色"
- 查看角色特征和翻译建议

### 4. 字符串翻译
- 切换到"字符串翻译"标签页
- 翻译菜单选项、按钮等 UI 文字

### 5. 对话翻译
- 切换到"对话翻译"标签页
- 使用搜索和筛选功能
- 点击"AI翻译"或"全部翻译"

### 6. 导出游戏
- 切换到"导出游戏"标签页
- 点击"开始导出"
- 导出目录：`projects/项目名/output/`

## 📁 项目结构

```
renpy-translator/
├── src/
│   ├── ui_nicegui.py          # NiceGUI 前端界面
│   ├── translator.py          # AI 翻译器
│   ├── renpy_parser.py        # Ren'Py 脚本解析器
│   ├── rpa_extractor.py       # RPA 文件解包器
│   ├── sdk_manager.py         # Ren'Py SDK 管理器
│   ├── project_manager.py     # 项目管理器
│   └── config_manager.py      # 配置管理器
├── projects/                  # 项目目录
│   └── 项目名/
│       ├── project.json       # 项目配置
│       ├── game/              # 解包后的游戏文件
│       └── output/            # 导出目录
├── config/                    # 配置目录
│   └── models.json            # AI 模型配置
├── run.py                     # 启动脚本
└── pyproject.toml             # 项目配置
```

## 🔧 技术栈

- **前端**: [NiceGUI](https://nicegui.io/) (Vue.js + Quasar)
- **后端**: Python 3.12+
- **AI**: OpenAI 兼容接口
- **SDK**: Ren'Py SDK
